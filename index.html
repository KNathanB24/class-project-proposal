Class Project Proposal

  1. Product Name: Projectile Motion Interactive Physics Simulator
  2. Team Members: Kyle Bledsoe
  3. Application Overview: A web based physics simulator that will allow students to practice understanding the concept of projectile motion. Students will be able to adjust physical parameters like gravity, height, angles, mass, and much more, and will be able to see a visual representation of the affects these variables have on a projectile motion system.
  4. Technology Overview: HTML and CSS will be used to add visual flair to the web page, while Javascript will be used for making the site interactive. Python may also be used in the background to work with complex calculations.
  5. Proposal Last Updated: 3/26/2026
  6. Feature List
    • User imput features, like sliders to adjust different values
    • A visual graph that changes with different imputs, and can be paused and replayed
    • Calculated values given to user
    • A section that teaches students the physics and math behind the simulation
  7. Future Feature List
    • 3D Simulations
    • Targets/Challenge mode
    • Multiple projectiles + collisions
    • Additional graphing for potential energy, kinetic energy, force, momentum, etc.
